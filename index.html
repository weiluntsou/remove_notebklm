PDF 轉可編輯 PPTX

上傳任意 PDF，AI 自動辨識每頁文字佈局，輸出每個文字都是真正可編輯文字框的 PowerPoint 檔案

🤖
AI 驅動，100% 本地處理： 使用 Google Gemini Vision 辨識文字位置與樣式，所有 PDF 渲染均在瀏覽器本地完成，檔案不會上傳至任何伺服器（除 Gemini API 外）。
📖 使用說明 ▼
  1. 輸入您的 Google Gemini API Key（取得 API Key）。
  2. 選擇 AI 模型（建議使用 Gemini 2.5 Flash 以達到最佳速度與精確度）。
  3. 將 PDF 檔案拖曳到下方上傳區域，或點擊選擇檔案。
  4. 點擊「開始轉換」，等待 AI 逐頁分析（每頁約 5-15 秒）。
  5. 轉換完成後，下載可編輯的 .pptx 檔案。
